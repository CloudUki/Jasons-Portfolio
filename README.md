# Jasons-Portfolio

This portfolio is for a friend of mine whom asked me to help design and develop a website for him. 

# Tech Stack
- Framework: React / TypeScript
- Styling: Tailwind CSS
- Run Time: NextJS

# Start Up Guide 
1. Copy and clone the repository on your computer
2. Once cloned, cd into the my-portfolio folder
```
cd my-portfolio
```
3. Once you are in the my-portfolio folder, run the developer instance
```
npm run dev
```
4. Copy the local url that is posted in the terminal into a browser you enjoy using
